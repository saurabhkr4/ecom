# ecom

To run the project:

1. clone the repo
2. open the folder in vs code
3. open terminal
4. cd frontend
5. npm install
6. open another terminal
7. cd backend
9. npm install
10. npm start server
11. now go back to first terminal
12. npm start
13. Open any browser and go to http://localhost:3000/login     (Don't go to http://localhost:3000)
